# isis
Insert a scenario into a script.

## メモ
### 仕様関係
シナリオ内のi番目の文章を、
スクリプトのi番目のシナリオタグの位置に
挿入する。

シナリオ内の空行は
次の文章を次のシナリオタグの位置に
挿入することを表す。

行頭の文字列でコマンドを判別する。
どのコマンドにも当てはまらなかった場合、
デフォルトのコマンドに判別される。

コマンドは設定ファイルで変更・追加できるようにする。

### 実装関係
最初にシナリオを解析して
コマンド別にキューに保存する。
次にスクリプトを解析して、
コマンドを見つけたら
そのコマンドのキューからデキューして、
得られたシナリオを挿入する。
